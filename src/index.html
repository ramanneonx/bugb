    ☁️ Google Drive Setup Guide

    To use the Auto-Backup feature, you need to create a free Google OAuth Client ID. Follow these simple steps:

    1. Google Cloud Console
    Go to Cloud Console and create a New Project (e.g. "BugbOS Backup").
    2. Configure OAuth Screen
    Go to APIs & Services > OAuth consent screen. Choose External and add your email. No other info needed.
    3. Create Credentials
    Go to Credentials > Create Credentials > OAuth client ID.
    • Application Type: Web Application
    • Authorized JavaScript origins: http://localhost (or leave blank if using desktop app)
    4. Copy Client ID
    Copy the Client ID string and paste it into the bugbOS settings. Click Save Key and then Authenticate.
    🛡️ PRIVACY: Your data is sent directly to your own Google Drive. bugbOS never sees your files.
